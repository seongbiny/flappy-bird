# Flappy Bird MVP PRD

## 1. 프로젝트 개요

Flappy Bird는 플레이어가 새 캐릭터를 조작해 파이프 장애물 사이를 통과하는 독립형 웹 미니게임이다.

플레이어는 클릭, 터치, 스페이스바 입력으로 캐릭터를 위로 점프시키고, 장애물이나 바닥에 닿지 않도록 버텨야 한다.

이 프로젝트는 `Vite`, `TypeScript`, `Pixi.js`, `Matter.js`를 사용한다.

* Pixi.js: 화면 렌더링
* Matter.js: 중력, 점프, 충돌 판정

## 2. MVP 목표

이번 MVP의 목표는 **플레이 가능한 가장 작은 버전의 Flappy Bird 게임을 완성하는 것**이다.

MVP에서 집중할 기능은 아래와 같다.

* 게임 시작
* 캐릭터 점프
* 중력 적용
* 파이프 장애물 생성 및 이동
* 충돌 판정
* 점수 증가
* 게임 오버
* 다시 시작

디자인, 애니메이션, 사운드, 랭킹, 최고 점수 저장은 MVP 범위에 포함하지 않는다.

## 3. 게임 규칙

* 게임은 시작 화면에서 시작한다.
* 플레이어는 새 캐릭터를 조작한다.
* 캐릭터는 Matter.js 중력에 의해 아래로 떨어진다.
* 클릭, 터치, 스페이스바 입력 시 캐릭터가 위로 점프한다.
* 파이프 장애물은 오른쪽에서 왼쪽으로 이동한다.
* 캐릭터가 파이프 사이를 통과하면 점수가 1점 증가한다.
* 캐릭터가 파이프, 바닥, 천장에 닿으면 게임 오버가 된다.
* 게임 오버 후 다시 시작할 수 있다.

## 4. 화면 구성

### 4.1 시작 화면

표시 요소:

* 게임 제목
* 간단한 조작 설명
* 시작 버튼

예시 문구:

```txt
Flappy Bird
클릭하거나 스페이스바를 눌러 날아오르세요!
```

### 4.2 게임 화면

표시 요소:

* 새 캐릭터
* 파이프 장애물
* 현재 점수
* 바닥

초기 버전에서는 이미지 에셋 없이 Pixi Graphics 도형으로 구현한다.

### 4.3 게임 오버 화면

표시 요소:

* Game Over 문구
* 최종 점수
* 다시 시작 버튼

예시 문구:

```txt
Game Over
Score: 7
다시 시작
```

## 5. 게임 상태

게임 상태는 3개만 사용한다.

```txt
ready       시작 전
playing     게임 진행 중
gameOver    게임 오버
```

### ready

* 시작 화면을 보여준다.
* 캐릭터와 파이프는 움직이지 않는다.
* 시작 버튼 또는 입력으로 `playing` 상태가 된다.

### playing

* Matter.js 물리 업데이트가 동작한다.
* 캐릭터에 중력이 적용된다.
* 입력 시 캐릭터가 점프한다.
* 파이프가 왼쪽으로 이동한다.
* 충돌 판정과 점수 계산을 수행한다.

### gameOver

* 게임 진행을 멈춘다.
* 최종 점수를 보여준다.
* 다시 시작 버튼을 누르면 게임을 처음부터 다시 시작한다.

## 6. 주요 오브젝트

### 6.1 Player

플레이어가 조작하는 새 캐릭터다.

* Matter.js Body로 충돌 영역을 관리한다.
* Pixi.js Graphics로 화면에 그린다.
* 입력 시 위쪽 방향으로 점프한다.
* 파이프, 바닥, 천장과 충돌하면 게임 오버가 된다.

### 6.2 Pipe

플레이어가 피해야 하는 장애물이다.

* 위쪽 파이프와 아래쪽 파이프 한 쌍으로 구성한다.
* Matter.js Body로 충돌 영역을 관리한다.
* Pixi.js Graphics로 화면에 그린다.
* 오른쪽에서 왼쪽으로 이동한다.
* 화면 밖으로 나가면 제거한다.

### 6.3 Ground

화면 하단의 바닥이다.

* Matter.js 정적 Body로 만든다.
* 플레이어가 바닥에 닿으면 게임 오버가 된다.

### 6.4 Ceiling

화면 상단의 천장이다.

* Matter.js 정적 Body로 만든다.
* 플레이어가 천장에 닿으면 게임 오버가 된다.

## 7. 점수 규칙

* 파이프 한 쌍을 통과하면 점수가 1점 증가한다.
* 한 파이프 쌍은 한 번만 점수를 줄 수 있다.
* 최고 점수 저장은 MVP에서 제외한다.

## 8. 기술 요구사항

* Vite
* TypeScript
* Pixi.js
* Matter.js
* ESLint
* Prettier

초기 버전에서는 React를 사용하지 않는다.

## 9. MVP 필수 기능

* 시작 화면 표시
* 게임 시작
* Matter.js Engine 생성
* Player Body 생성
* Player 점프 처리
* 중력 적용
* Pipe Body 생성
* Pipe 이동
* Player와 Pipe 충돌 감지
* Player와 Ground 충돌 감지
* Player와 Ceiling 충돌 감지
* 점수 증가
* 게임 오버 화면 표시
* 다시 시작
* Pixi.js 리소스 정리
* Matter.js Body 및 이벤트 정리

## 10. MVP 제외 기능

아래 기능은 MVP에서 만들지 않는다.

* 로그인
* 랭킹
* 서버 연동
* 최고 점수 저장
* 사운드
* 캐릭터 애니메이션
* 배경 스크롤
* 바닥 스크롤
* 난이도 증가
* 캐릭터 선택
* 모바일 최적화 고도화
* React
* Supabase

## 11. 성공 기준

MVP는 아래 조건을 만족하면 완료로 본다.

* `pnpm dev`로 게임이 실행된다.
* 시작 화면이 보인다.
* 시작 버튼 또는 입력으로 게임을 시작할 수 있다.
* 캐릭터가 중력에 의해 아래로 떨어진다.
* 클릭, 터치, 스페이스바 입력 시 캐릭터가 점프한다.
* 파이프가 오른쪽에서 왼쪽으로 이동한다.
* 파이프 사이를 통과하면 점수가 1점 증가한다.
* 파이프에 닿으면 게임 오버가 된다.
* 바닥 또는 천장에 닿으면 게임 오버가 된다.
* 게임 오버 화면에서 최종 점수를 볼 수 있다.
* 다시 시작 버튼으로 게임을 재시작할 수 있다.
* 콘솔에 치명적인 에러가 없다.
